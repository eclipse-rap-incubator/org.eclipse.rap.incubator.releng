To generate a p2 repository from this feature follow the following steps:
* open the deployable feature export wizard and select "org.eclipse.rap.incubator.feature" 
* in the "Destination" tab of the export wizard enter the location where you want to export
* in the "Options" tab select the options:
  - "Generate Source Bundles"
  - "Generate metadata repository"
  - "Categorize repository" and enter the location of "category.xml" (which is included into this feature)
